üîß Troubleshooting

  • Cover Generation Issues:
    • If the preview is not updating:
      • Check if all required fields are filled correctly
      • Try switching templates to refresh the preview
      • Clear your browser cache and reload the page
    • If the preview appears blank:
      • Ensure your browser is up to date
      • Try disabling any ad-blockers temporarily
      • Check your internet connection for logo loading
  • Download/Export Issues:
    • PDF not downloading:
      • Check if pop-ups are blocked in your browser
      • Try using a different browser (Chrome recommended)
      • Ensure you have enough storage space
    • Print quality issues:
      • Enable "Background Graphics" in print settings
      • Set paper size to A4
      • Choose "High Quality" in print settings
  • Data Management Issues:
    • If saved data isn't loading:
      • Check if your browser allows local storage
      • Ensure you haven't cleared browser data recently
      • Try saving the data again
    • Auto-save not working:
      • Check your browser's privacy settings
      • Ensure you're not in private/incognito mode
      • Try manual save as a temporary solution
  • Display/Layout Issues:
    • Text overflow problems:
      • Use the font size controls to adjust text size
      • Try a different template
      • Keep input text length reasonable
    • Mobile display issues:
      • Ensure your device is in portrait orientation
      • Try zooming out if content is cut off
      • Use the mobile-optimized templates
Need More Help? If you're still experiencing issues, try:
