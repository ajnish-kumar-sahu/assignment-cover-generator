## 🌟 Overview

**ScholarFlow** is a premium academic document creation and management suite that transforms boring assignments into professional, visually stunning documents.

Whether you're a student or researcher, ScholarFlow helps you:

- 🖼️ Design polished assignment covers
- 📋 Create structured indexes and tables
- 🗂️ Manage academic documents with ease
- 📤 Export professional PDFs instantly

## 🚀 Quick Start

### Prerequisites

- Node.js **18+**
- npm or yarn

### Installation

```bash
git clone https://github.com/ajnish-kumar-sahu/assignment-cover-generator.git
cd assignment-cover-generator
npm install
npm run dev
```

👉 App runs at: **[http://localhost:5173](http://localhost:5173)**

---

## 🔐 Environment Variables

Create a `.env` file in the project root:

```env
VITE_APP_NAME=ScholarFlow
VITE_API_URL=http://localhost:5000
```

## 💻 Tech Stack

### Frontend
- **React 19** — UI framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Utility-first styling

### UI & Animation

- **Lucide Icons** — Clean icon set

### Export Engine
- **html2canvas** — DOM to canvas rendering
- **jsPDF** — PDF generation

### State Management
- **Zustand** — Lightweight global state

### Build Tools

- **ESLint** — Code quality linting
