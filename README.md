🌟 Overview

ScholarFlow is a premium academic document creation and management suite that transforms boring assignments into professional, visually stunning documents.

Whether you're a student or researcher, ScholarFlow helps you:

- Design assignment covers
- Create structured indexes
- Manage academic documents
- Export professional PDFs instantly

---

💡 Why ScholarFlow?

Most students struggle with:

- ❌ Poor formatting
- ❌ Time-consuming manual work
- ❌ Lack of design tools

✅ Solution

ScholarFlow provides:

- ⚡ Instant professional templates
- 🎨 Beautiful UI/UX
- 💾 Auto-save system
- 📤 Export in multiple formats

🚀 Quick Start

Prerequisites

- Node.js 18+
- npm / yarn

Installation

git clone https://github.com/ajnish-kumar-sahu/assignment-cover-generator.git
cd assignment-cover-generator
npm install
npm run dev

👉 App runs at: "http://localhost:5173"

---

🔐 Environment Variables

Create ".env" file:

VITE_APP_NAME=ScholarFlow
VITE_API_URL=http://localhost:5000
